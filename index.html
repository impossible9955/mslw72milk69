# วันที่ ประเภท รายละเอียด เข้า (กล่อง) ออก (กล่อง) คงเหลือ (กล่อง)

☁️ ซิงก์ข้อมูลออนไลน์ (Firebase Realtime DB)
🔥 ใช้ Firebase เพื่อซิงก์ข้อมูลแบบ realtime ระหว่างหลายเครื่องพร้อมกัน ไม่ต้องส่งไฟล์
💡 วิธีตั้งค่า: ไปที่ Firebase Console → สร้างโปรเจค → Realtime Database → Copy URL และ API Key มาใส่ข้างบน แล้วตั้งค่า Rules เป็น read/write: true
🆕 ปุ่ม "อัปโหลด/ดึงข้อมูล (ทั้งหมด)" ใช้สำหรับซิงก์ ห้องเรียน/สต็อก/การตั้งค่า เท่านั้น แนะนำให้กดแบบ manual เมื่อจำเป็น — ส่วนข้อมูล การเช็คดื่มนมรายวัน ของครู จะถูกส่งขึ้น Cloud อัตโนมัติทันทีทุกครั้งที่บันทึก (เฉพาะรายการที่เปลี่ยน ไม่ใช่ทั้งฐานข้อมูล) อยู่แล้วโดยไม่ต้องกดปุ่มนี้ Auto-Sync ด้านบนใช้สำหรับ "ดึง" ข้อมูลการเช็คของครูล่าสุดเข้ามาทุก 3 นาทีเท่านั้น
🔄 กำลังซิงก์...
🧮 สต็อกย่อยรายห้องเรียน
💡 ระบบเปลี่ยนมาคิดสต็อกแยกเป็นรายห้องเรียน — "จ่ายนมให้ห้องเรียน" คือจุดที่ย้ายนมจากส่วนกลางไปไว้ที่ห้อง ส่วนการเช็คดื่มนมรายวัน/นมค้างรายสัปดาห์จะหักจากสต็อกของห้องนั้นเท่านั้น ไม่กระทบห้องอื่นหรือสต็อกกลางอีกต่อไป ปุ่มด้านล่างใช้คำนวณยอดสต็อกย่อยของทุกห้องใหม่จากประวัติทั้งหมด (ทำซ้ำได้ ไม่ทำให้ข้อมูลเดิมเสียหาย) — แนะนำให้กดหลังการอัปเดตระบบครั้งแรก หรือถ้าสงสัยว่ายอดห้องใดไม่ตรง
💡 ถ้าพบว่า "ตัวเลขสต็อกบนหน้าจอไม่ตรงกับประวัติรับ/จ่ายที่มีอยู่จริง" ให้กดปุ่ม 🔧 ซ่อมสต็อกทั้งหมด — ระบบจะคำนวณทั้งสต็อกกลางและสต็อกย่อยรายห้องใหม่จากประวัติทั้งหมดทันที ไม่ต้อง export/import ไฟล์ใดๆ และไม่ลบประวัติเดิม ทำซ้ำได้ปลอดภัย

💾 สำรองข้อมูล
💡 "ล้างข้อมูลทั้งหมด" ด้านบน = รีเซ็ตเฉพาะรับนม/จ่ายนมให้ห้องเรียน/ห้องเรียน (เก็บการตั้งค่าไว้) — ถ้าต้องการล้าง แคชในเครื่องนี้แบบครบทุกส่วน (รวมเช็คดื่มนม/นมค้าง/นมย้อนหลัง/นมปิดเทอม) หรือ ลบฐานข้อมูลทั้งหมดรวม Cloud ดูหัวข้อ "Danger Zone" ด้านล่าง

🧹 ล้างแคชเครื่องนี้ทั้งหมด
ลบข้อมูลที่เก็บไว้ในเบราว์เซอร์นี้ทั้งหมด (รับนม/จ่ายนม/เช็คดื่มนม/นมค้าง/นมย้อนหลัง/นมปิดเทอม/session) โดยไม่กระทบข้อมูลบน Cloud — ใช้แก้ปัญหาเครื่องนี้ค้าง/ตัวเลขประหลาด แล้วดึงข้อมูลจาก Cloud กลับมาใหม่ได้
💣 ลบฐานข้อมูลทั้งหมด (รวม Cloud)
ลบข้อมูลทั้งหมดอย่างถาวร ทั้งในเครื่องนี้และบน Cloud (Firebase) — กระทบทุกอุปกรณ์/ครูทุกคนที่ใช้ระบบนี้อยู่ ต้องพิมพ์ข้อความยืนยันก่อนเริ่ม ใช้เมื่อต้องการเริ่มต้นระบบใหม่ทั้งหมดเท่านั้น
