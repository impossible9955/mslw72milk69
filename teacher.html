👤 ข้อมูลครูประจำชั้น (แก้ไขได้)
📌 ข้อมูลนี้ใช้ฐานข้อมูลเดียวกับระบบผู้ดูแล (index.html) — เมื่อบันทึก ชื่อครูประจำชั้นของห้องนี้จะถูกอัปเดตทันที และจะถูกใช้แสดงอัตโนมัติในหน้า "เช็คดื่มนมรายวัน"
📖 วิธีใช้งาน
1. Firebase URL ถูกตั้งค่าให้ตรงกับระบบหลักไว้แล้ว (ใช้ฐานข้อมูลเดียวกัน) — ไม่ต้องตั้งค่าเพิ่ม
2. กด "โหลดห้องเรียน" → เลือกห้อง → ใส่รหัสผ่าน → เข้าระบบ
3. เช็คดื่มนมรายวัน: เลือกวันที่ → กดปุ่ม ✓/✗ แต่ละคน → บันทึก
4. ข้อมูลซิงก์ Firebase ทันที (milkApp/mcAttendance) ผู้ดูแลระบบเห็นข้อมูลเมื่อเข้าหน้าภาพรวมการดื่มนมหรือกด "ซิงก์ข้อมูลครูล่าสุด"
5. สามารถถ่ายรูปและเซ็นชื่อครูได้ทุกครั้งที่บันทึก
